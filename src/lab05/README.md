# ЛР-5 — Функции как аргументы. Стратегии и делегаты

## Цель работы

Освоить передачу функций как аргументов в другие функции и методы, научиться применять встроенные функции высшего порядка (`map`, `filter`, `sorted`), понять концепцию паттерна «Стратегия» и реализовать его на Python, освоить `lambda`-выражения и их практическое применение, интегрировать функциональный стиль с объектно-ориентированным кодом из предыдущих лабораторных работ.

---

## Предметная область

**Игровая логика / RPG**

Базовая сущность: **`Player`** (Игрок) → иерархия `Warrior`, `Mage`, `Archer`

---

## Реализованные функции и стратегии

### Стратегии сортировки (оценка 3)

| Функция | Описание | Возвращает |
|---------|----------|------------|
| `by_name(player)` | Сортировка по имени | `str` |
| `by_level(player)` | Сортировка по уровню | `int` |
| `by_power(player)` | Сортировка по силе | `float` |
| `by_health(player)` | Сортировка по здоровью | `float` |
| `by_name_then_level(player)` | Многоуровневая сортировка | `tuple` |

### Функции-фильтры (оценка 3)

| Функция | Описание |
|---------|----------|
| `is_alive_filter(player)` | Только живые игроки |
| `is_high_level_filter(player)` | Уровень >= 5 |
| `is_warrior_filter(player)` | Только воины |
| `is_healthy_filter(player)` | Здоровье > 50 |

### Фабрики функций (оценка 4)

| Функция | Описание | Пример |
|---------|----------|--------|
| `make_level_filter(min_level)` | Создаёт фильтр по уровню | `filter_5 = make_level_filter(5)` |
| `make_health_filter(min_health)` | Создаёт фильтр по здоровью | `filter_70 = make_health_filter(70)` |
| `make_damage_multiplier(mult)` | Создаёт функцию нанесения урона | `apply_x2 = make_damage_multiplier(2)` |

### Callable-объекты (паттерн Стратегия) (оценка 5)

| Класс | Описание | Метод `__call__` |
|-------|----------|------------------|
| `DiscountStrategy` | Стратегия начисления опыта | Добавляет опыт с бонусом класса |
| `HealthRestoreStrategy` | Стратегия лечения | Восстанавливает здоровье |
| `PowerBoostStrategy` | Стратегия усиления | Добавляет бонусный опыт |

### Функции для `map` (оценка 4)

| Функция | Вход | Выход |
|---------|------|-------|
| `to_info_string(player)` | Игрок | Информационная строка |
| `extract_name_and_level(player)` | Игрок | Словарь с данными |
| `to_tuple(player)` | Игрок | Кортеж (имя, уровень, сила) |

### Методы коллекции (оценка 4-5)

| Метод | Описание |
|-------|----------|
| `sort_by(key_func, reverse)` | Сортировка по переданной стратегии |
| `filter_by(predicate)` | Фильтрация по переданному условию |
| `apply(func)` | Применение функции ко всем элементам |
| `map_to(transform_func)` | Преобразование через `map` |
